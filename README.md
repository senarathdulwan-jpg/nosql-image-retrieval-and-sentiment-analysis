# NoSQL Image Retrieval & Sentiment Analysis

This project focuses on building two NoSQL databases using MongoDB to handle unstructured data.

The first part implements a Content-Based Image Retrieval (CBIR) system, where images are processed, annotated, and stored with extracted features for similarity-based retrieval.

The second part focuses on sentiment analysis using customer reviews, where text data is cleaned, vectorized (BoW & TF-IDF), and labeled using sentiment analysis techniques.

## Tools Used
- Python (OpenCV, NLTK, Scikit-learn)
- MongoDB
- JSON

This project demonstrates how unstructured data such as images and text can be processed and stored efficiently for real-world applications.
